jquery.blockify

This entire paragraph is made clickable by jquery.blockify. By default the href from last anchor tag is used to determine the destination of a click. In this case I've overridden the default and the first anchor is used instead. All links within the selected element still functional as standard links and the plugin does nothing if text has been selected. giddyup. also, this is an internal link and another one for good measure

This paragraph is not selected and is therefore left untouched.

This paragraph is selected but is ignored because it contains no links.

You can easily ignore certain elements of your choosing as well (using the ignore option).







If you click anywhere else in this paragraph you will link to google.

uses

this demo uses paragraph tags, but this plugin can be on any element. i've used it mostly on <li> tags that contain an article snippet with a read more link.

options

  • selector:'a:last' // selector to specific anchor (or other element), in case there's more than one!
  • cursor: 'pointer' // sets css cursor value. set to false to do nothing
  • hoverClass:'hover' // class attached to element on hover
  • externalClass:'ext' // if target anchor has this class, link will be opened in a new window
